SOS Game ->
It is  a simple 2 players game.
* Each player can either mark "S" or "O" on the board.
* The player who can get a "SOS" conbination first will win.

TIC-TAC-TOE Game
* The player to get a series of X or O will win.
Enjoy !!!!!
